Limbajul HTML

HTML(HyperText Markup Language) este un limbaj de marcare utilizat pentru a crea pagini web

un exemplu de adresa web il puteti gasii la adresa http://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png.

Imaginea are atributul height 147

Tag-urile html cele mai utilizate in realizarea tabelelor sunt

table Defineste un tabel
tr Defineste o linie intr-un tabel
td Defineste o celula intr-un tabel

Observatie: pentru a pune contur unui tabel se poate (dar nu se recomanda) a se pune atributul border="1".

La validare va da eroarea: The border attribute on the table element is obsolete.

Formularele se pot creea cu ajutorul tag-ului form care poate contine tag-urile : input, textarea, label, select, option button.

pentru mai multe detalii consultati cursul sau tutorialul HTML Forms and Inputs de la adresa www.w3schools.com/html/html_forms.asp.
